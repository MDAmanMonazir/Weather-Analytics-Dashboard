

# Weather Analytics Dashboard 🌦️

A comprehensive data visualization project that tracks and analyzes real-time weather patterns, climate trends, and atmospheric KPIs using the WeatherAPI.

## 📌 Overview
This project focuses on transforming raw meteorological data into actionable insights. By connecting directly to live APIs, the dashboard provides a dynamic view of weather conditions, historical trends, and predictive analytics for various geographical locations.

## 🛠️ Technologies Used
* **Data Source:** [WeatherAPI.com](Free Weather API - WeatherAPI.com https://share.google/90V7zeIK0BigDDUys) (Real-time & Forecast data)
* **Visualization:** Power BI Desktop
* **Data Transformation:** Power Query (M Language)
* **Analytics:** DAX (Data Analysis Expressions)
* **Connectivity:** Web API Integration

## 🚀 Key Features
* **Real-time Integration:** Automated data fetching from WeatherAPI to keep the dashboard updated.
* **KPI Tracking:** High-level metrics for Temperature, Humidity, UV Index, and Wind Speed.
* **Trend Analysis:** Interactive line graphs showing temperature fluctuations over time.
* **Geospatial Mapping:** Visual representation of weather conditions across different regions.
* **Advanced DAX:** Custom measures for calculating averages, variance in rainfall, and temperature conversions.

## 📊 Dashboard Components
* **KPI Cards:** Instant visibility into current weather essentials (e.g., "Feels Like" temp, Cloud Cover %).
* **Time-Series Graphs:** Detailed views of hourly and daily weather changes.
* **Power Query Transformation:** Cleaned and structured JSON responses from the API into a relational model suitable for analysis.
* **Interactive Filters:** Slicers for Location, Date Range, and Weather Conditions (Sunny, Cloudy, Rainy).

## ⚙️ How to Setup
1.  **Get API Key:** Sign up at [WeatherAPI.com](https://www.weatherapi.com/) to get your free API key.
2.  **Power BI Configuration:**
    * Open the `.pbix` file.
    * Go to `Transform Data` -> `Data Source Settings`.
    * Replace the API URL with your specific location and API key.
3.  **Refresh:** Click on the 'Refresh' button in the Home tab to pull the latest data.
